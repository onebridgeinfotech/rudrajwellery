=== Jwellery Jewelry ===
Standalone WooCommerce theme for imitation jewelry stores.
Inspired by https://ramyanagendra.com/

Requires: WordPress 6+, WooCommerce, Jewelry UPI Store plugin

Install:
1. Upload jwellery-jewelry.zip via Appearance → Themes
2. Upload jewelry-upi-store.zip via Plugins
3. Import sample-data/products-sample.csv

See docs/HOSTINGER-INSTALL.md in project root.
