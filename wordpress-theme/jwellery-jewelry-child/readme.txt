=== Jwellery Jewelry Child ===

Child theme for Kadence. Do not install without Kadence parent.

Installation:
1. Appearance → Themes → Add New → search "Kadence" → Install → Activate
2. Plugins → Upload jewelry-upi-store.zip (NOT this folder as a plugin)
3. Appearance → Themes → Add New → Upload jwellery-jewelry-child.zip → Activate

Do NOT upload the whole "jwellery ecommerce" project folder as a theme.
