# UI enhancements (theme v2.4.0)

## After uploading theme

1. **Appearance → Store Setup → Full setup** (or run each button once):
   - Enable INR + promo codes
   - Import product images
   - Full setup also assigns **category thumbnails** from products

2. **Appearance → Customize**
   - **Store UI:** brand name, WhatsApp number (10 digits), announcement messages (`|` separated), newsletter heading
   - **Homepage Hero:** up to 3 slide images (or uses bundled default)
   - **Social Links:** Facebook, Instagram, YouTube
   - **Site Identity:** upload logo + site icon (favicon)

## What’s included

| Feature | Description |
|---------|-------------|
| Hero slideshow | 3 customizable images + default jewelry photo |
| Rotating announcement bar | Shipping, promo, UPI messages |
| Mobile search | Search icon opens panel on phone |
| Cart icon + badge | Header cart with count |
| Floating WhatsApp | Fixed button when number set in Customizer |
| Homepage | Hides sold-out products from carousels |
| Category images | Auto-set from first product per category on setup |
| Product cards | Hover zoom, 2-line titles, add-to-cart toast |
| Carousels | Dots, auto-scroll (Customizer toggle), arrows inside |

| Product page | Trust row, WhatsApp share, sticky add-to-cart (mobile) |
| Cart | Login notice for guests |
| Shop | 2 columns mobile, sort toolbar |
| Testimonials + WhatsApp CTA | Bottom of homepage |
