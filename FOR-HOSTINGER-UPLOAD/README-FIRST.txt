================================================================================
  DO NOT UPLOAD "jwellery ecommerce.zip" OR "jwellery ecommerce (2).zip"
  That is the WRONG file — WordPress will say "missing style.css"
================================================================================

UPLOAD ONLY THESE 2 FILES TO WORDPRESS:

  1) jwellery-jewelry.zip
     -> WordPress: Appearance -> Themes -> Add New -> Upload Theme

  2) jewelry-upi-store.zip
     -> WordPress: Plugins -> Add New -> Upload Plugin

ORDER:
  1. Install WordPress on Hostinger (hPanel Auto Installer)
  2. Install WooCommerce plugin (search in Plugins -> Add New)
  3. Upload jwellery-jewelry.zip as THEME
  4. Upload jewelry-upi-store.zip as PLUGIN
  5. WooCommerce -> Settings -> Payments -> Manual UPI -> add your UPI ID

Full guide: docs\HOSTINGER-INSTALL.md
